carte.js

WebGL Visualization for Google Maps

Fast

Renders points, polygons and rasters multiple times faster than canvas. Minimal draw calls by using texture atlases generated on the fly to render objects by batch.

Supports tiling

Automatically renders tiles from either vector or image sources. Currently supports sources from /z/x/y tile servers with either GeoJSON or PNG formats. Caches each tile so dragging and panning is smooth once all tiles are loaded.

Extendable

Will be adding more file formats possibly including MVT, TopoJSON and other custom binary formats.
